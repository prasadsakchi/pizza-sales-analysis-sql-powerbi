# Pizza Sales Analysis

## Project Overview
This project analyzes pizza sales data to understand revenue trends, customer ordering behavior, and product performance.

## Business Objective
- Analyze overall sales and revenue performance  
- Identify peak ordering periods  
- Understand customer preferences by pizza category and size  
- Identify top and bottom selling pizzas  

## Dataset
- Pizza sales transactional dataset  
- Includes order date, pizza name, category, size, quantity, and price  

## Tools Used
- SQL (Data Analysis & KPI Calculation)
- Power BI (Dashboard & Visualization)

## KPIs Developed
- Total Revenue  
- Average Order Value  
- Total Orders  
- Average Pizzas per Order  

## Analysis Performed
- Daily and monthly order trends  
- Revenue contribution by pizza category  
- Revenue contribution by pizza size  
- Top and bottom selling pizzas  

## Key Insights
- Higher order volumes observed during weekends  
- Medium and large pizzas contribute higher revenue  
- A few top pizzas generate a major share of total sales  

## Key Learnings
- Writing SQL queries for business-driven questions  
- Understanding customer purchase behavior  
- Presenting insights using dashboards
